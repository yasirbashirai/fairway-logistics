# Fairway Logistics LLC - Enterprise Website

A conversion-focused, SEO-optimized, enterprise-grade website for Fairway Logistics LLC — an asset-based logistics company serving the Gulf Coast region from Mobile, Alabama.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Animations:** CSS Animations
- **Forms:** React Hook Form + Zod validation
- **Deployment:** Vercel-ready

## Features

- 30 fully optimized pages (services, regional, industry, resources)
- Multi-step freight quote form on every major page
- SEO-optimized with meta tags, Open Graph, and JSON-LD structured data
- Responsive design (mobile-first)
- Sticky CTA bar for lead capture
- Interactive FAQ accordions
- Animated metrics counters
- Mega dropdown navigation
- Carrier partner application form
- Contact form with Google Maps integration

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to preview.

## Build for Production

```bash
npm run build
npm start
```

## Deploy

Deploy to Vercel:
```bash
npx vercel
```
